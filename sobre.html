Nossa trajetória de sucesso

  • Surgimento das Sweetie Girls

    Nossa squad é composta por integrantes muito unidas desde o início do curso, cada uma com sua individualidade e habilidade marcante, contribuindo para o sucesso do projeto, sendo elas: Ana Julia Oliveira, Larissa Abreu, Raissa Pinheiro e Vitória de Queiroz.

    11 de maio

  • Início do projeto Sweetie Cake

    Depois de longos períodos de discussões no grupo, chegamos ao consensso de que uma confeitaria seria a melhor opção para trabalharmos, agradando a todas. Inicializamos na parte de design do site com HTML e CSS. Cada uma responsável pela execução de determinada área do projeto de acordo com suas habilidades.

    13 de maio

  • Separação do trabalho

    Como dito anteriormente, cada uma ficou com uma parte do projeto, podendo, obviamente, prestar serviços para as outras colegas, afim de agilizar o processo. No design, Vitória ficou como responsável, com auxílio de Larissa, responsável pelo relatório e pesquisas transversais, já em Java, Ana e Raissa mostraram suas competências.

    16 de maio

  • Execução do projeto

    Nos baseamos nos conhecimentos adquiridos no curso para iniciar o projeto, logo utilizamos dos infinitos recursos provenientes da linguagem de programação JAVA, assim como elaboramos uma interface mais interativa através do uso do HTML e CSS para alcançar o melhor resultado.

    17 de maio

  • Interface Gráfica

    Utilizando o HTML conseguimos obter excelentes resultados quanto a escolha da paleta de cores voltadas para o bege e marrom, e juntamente com o CSS, desenvolvemos diversas animações para uma interface interativa.

    19 de maio

  • Cadastrando nossos clientes

    Usufruindo daquilo que o JAVA nos fornece,desenvolvemos um programa capaz de cadastrar o pedido de nossos clientes, utilizando matriz dinâmica (índice), polimorfismo e também exploramos o que as classes JPanel e JFrame nos traz.

    20 de maio
